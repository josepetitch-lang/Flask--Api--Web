Guía de Referencia Técnica: Flask

Estructura modular optimizada para el desarrollo de microservicios y APIs estructuradas empleando Python.

1. Inicialización del Entorno

Antes de estructurar la aplicación, se debe garantizar el aislamiento de dependencias utilizando un entorno virtual limpio:

# Crear entorno virtual en la carpeta raíz
python -m venv venv

# Activar el entorno (
venv\Scripts\activate

# Instalación de dependencias core
pip install Flask flask-migrate flask-sqlalchemy

2. Servidor Mínimo Estándar

Estructura base para levantar un servidor de escucha local. El punto de entrada se inicializa instanciando la clase principal de Flask:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Servidor Flask corriendo correctamente."

if __name__ == '__main__':
    # Modo debug activo para recarga automática en desarrollo
    app.run(debug=True, port=5000)

3. Enrutamiento Dinámico

Flask mapea URLs a funciones mediante decoradores. Es posible capturar variables directamente desde la ruta especificando el tipo de dato requerido:

@app.route('/usuario/<int:user_id>')
def obtener_usuario(user_id):
    return f"Buscando los registros asociados al ID numérico: {user_id}"

4. Construcción de Endpoints API (REST)

Para comunicar servicios externos (por ejemplo, peticiones desde Postman o interfaces web), las respuestas se retornan serializadas en formato estructurado JSON:

from flask import jsonify, request

@app.route('/api/v1/recurso', methods=['POST'])
def crear_recurso():
    # Capturar el cuerpo de la petición HTTP recibida
    datos = request.get_json()
    
    respuesta = {
        "estado": "exitoso",
        "mensaje": "Recurso procesado correctamente",
        "datos_recibidos": datos
    }
    return jsonify(respuesta), 201

5. Control de Base de Datos y Migraciones PRO

Manejo de persistencia relacional acoplando el ORM y la herramienta de control de versiones de bases de datos:

from flask_sqlalchemy import SQLAlchemy
from flask_migrate import Migrate

db = SQLAlchemy(app)
migrate = Migrate(app, db)

# Comandos esenciales para la consola de desarrollo:
# 1. flask db init       -> Crea la carpeta base de migraciones
# 2. flask db migrate -m -> Detecta cambios en los modelos y genera el script
# 3. flask db upgrade    -> Aplica los cambios estructurales en la BD real
