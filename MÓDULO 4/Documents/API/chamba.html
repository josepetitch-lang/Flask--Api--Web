Construcción de Endpoints en Flask

Para interactuar como una API, Flask debe retornar diccionarios de Python, los cuales son serializados automáticamente a formato JSON.

Ejemplo Técnico Básicos:

from flask import Flask, jsonify, request

app = Flask(__name__)

# GET Obtener información
@app.route('/api/usuario', methods=['GET'])
def obtener_usuario():
    datos = {"nombre": "Andrew", "rol": "Developer"}
    return jsonify(datos), 200

# POST Crear o enviar información
@app.route('/api/usuario', methods=['POST'])
def crear_usuario():
    info_recibida = request.json
    return jsonify({"mensaje": "Usuario creado", "data": info_recibida}), 201

if __name__ == '__main__':
    app.run(debug=True)
