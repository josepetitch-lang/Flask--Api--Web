Flask API Documentation

Guía de referencia rápida para interactuar con los endpoints del backend desde JavaScript.

GET /api/v1/status

Retorna el estado actual del servidor y la sincronización del sistema.

Respuesta Exitosa (JSON):
{
    "status": "online",
    "timestamp": 1718568000,
    "version": "1.0.2"
}
POST /api/v1/action

Envía datos estructurados para procesar una nueva transacción o publicación.

Cuerpo de la Petición (Required JSON):
{
    "user_id": "panas_07",
    "type": "standard",
    "payload": {
        "score": 2400
    }
}
Respuesta Exitosa (JSON - 201 Created):
{
    "success": true,
    "message": "Registro procesado correctamente",
    "id": "TX-99821"
}
