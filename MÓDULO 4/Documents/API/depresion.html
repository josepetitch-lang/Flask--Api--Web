Consumo de APIs con JavaScript (Fetch API)

Estructura de una Petición GET:

const url = 'http://127.0.0.1:5000/api/usuario';

fetch(url)
    .then(respuesta => {
        if (!respuesta.ok) {
            throw new Error('Error en la conexión con el servidor');
        }
        return respuesta.json(); // Convierte la respuesta JSON a objeto JS
    })
    .then(data => {
        console.log('Datos del backend:', data);
        // Aquí manipulas el HTML (DOM) para mostrar los datos en pantalla
        document.getElementById('nombre').innerText = data.nombre;
    })
    .catch(error => {
        console.error('Hubo un problema con la petición:', error);
    });
