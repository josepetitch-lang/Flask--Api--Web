# Conceptos básicos

-- **Flask**:
* **¿Qué es?:** Un micro-framework escrito en Python diseñado para crear aplicaciones web de forma rápida y con la menor cantidad de líneas de código posible.
* **¿En qué se usa?:** Se utiliza para levantar el servidor backend de una aplicación. Es el encargado de gestionar las rutas (ej. `/inicio`, `/usuarios`), conectarse a bases de datos (usando herramientas como *Flask-SQLAlchemy*) y procesar la lógica de negocio antes de enviar una respuesta al cliente.


-- **API**:
* **¿Qué es?:** Un conjunto de reglas y definiciones que permite que dos aplicaciones o componentes de software se comuniquen entre sí. En el entorno web, solemos trabajar con **APIs REST**.
* **¿En qué se usa?:** Sirve como "puente". Por ejemplo, tu backend en Flask puede exponer una API para que una aplicación móvil, un frontend separado o herramientas de prueba como **Postman** puedan pedir o guardar datos (en formato JSON) sin necesidad de recargar la página entera

-- **Html**: 
* **¿Qué es?:** El lenguaje de marcado estándar para la creación de páginas web. No es un lenguaje de programación, sino la estructura esquelética del sitio.
* **¿En qué se usa?:** Se usa para definir el contenido de la página: párrafos, títulos (`<h1>`), contenedores (`<div>`), formularios, imágenes y la importación de otros recursos.

-- **Css**:
* **¿Qué es?:** El lenguaje de diseño gráfico utilizado para describir la presentación de un documento estructurado en HTML.
* **¿En qué se usa?:** Se encarga de toda la estética visual. Controla los colores, las tipografías, el espaciado (márgenes/padding), las animaciones y el diseño responsivo (que se adapte a celulares y computadoras).

-- **Javascript**:
* **¿Qué es?:** El lenguaje de programación por excelencia de la web. Se ejecuta directamente en el navegador del usuario (frontend).
* **¿En qué se usa?:** Aporta dinamismo e interactividad. Se usa para validar formularios antes de enviarlos, realizar peticiones asíncronas a una API (usando `fetch`) sin recargar la pantalla, crear efectos visuales interactivos y manejar eventos del usuario (clics, pulsaciones de teclas)

-- **Web**:
* **¿Qué es?:** El sistema de gestión de información que utiliza internet como medio de transmisión (HTTP/HTTPS).
* **¿En qué se usa?:** Es el ecosistema donde converge todo lo anterior. Conecta a los clientes (navegadores) con los servidores, permitiendo el funcionamiento de plataformas de streaming, redes sociales, comercios electrónicos y páginas personales.
